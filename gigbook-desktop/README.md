# GigBook Desktop

Ejecutable de escritorio para GigBook Server.

## Uso rapido

1. Descarga `GigBook-Server-Portable.exe`
2. Doble clic para ejecutar
3. Listo! El servidor arranca automaticamente

## Desarrollo

```bash
cd gigbook-desktop
npm install
npm start        # Modo desarrollo
npm run build    # Crear ejecutable
```

## Caracteristicas

- Interfaz grafica para controlar el servidor
- Minimizar a bandeja del sistema
- Copiar URL y token con un clic
- Consola de debug integrada
- Auto-inicio del servidor
